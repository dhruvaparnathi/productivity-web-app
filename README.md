# Productivity Web App

A modern productivity web application built using HTML, CSS, and JavaScript.

It includes task management, daily planning, motivational quotes, and a Pomodoro timer — all in a clean glassmorphism interface.

---

## Features

- Drag & Drop To-Do List (LocalStorage supported)
- Daily Planner with auto-generated hourly slots
- Motivational Quotes API integration
- Pomodoro Timer (Focus & Break cycle)
- Dynamic Theme Switching
- Persistent Data Storage using LocalStorage

---

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript
- LocalStorage API
- Fetch API
